# Commodity_Calculator_Service

# Goals
- 꽃 관련 자영업자들이 편리하게 원자재 비교 및 재고 관리 할 수 있게 함
- Flower-related self-employed people can conveniently compare commodities and manage inventory.

# Format
- 사진 기반으로 원자재 정보를 기록·관리하고, 선택한 원자재로 합계 원가를 계산하는 iOS 앱
- An iOS app that records and manages raw material information based on photos and calculates the total cost with selected raw materials.

# Targets: 
- Small-scale start-up user
- 재료 비용 관리 & 원가 계산이 필요한 사람 (People who need material cost management & cost calculation)

# Upgrade Point
- Existed app's error
- Required points
    - Adding picture
    - Typing Informations
      - Commodity name
      - Store name (Where user purchases)
      - Purchse value (How much it takes)
      - Inventory managing (How many metarials left)
    - Creating Information Card of specific metarial (Easy to Search)
    - Calculating Method
      - Sum of metarials that user choose manualy
